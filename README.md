🚗 UniRide – Open Source Campus Ride Sharing Platform
UniRide is an open-source ride sharing platform built specifically for college communities. It connects students travelling along similar routes, enabling them to share rides, split fuel costs, and reduce daily commuting expenses.
📖 Problem Statement
Many students commute daily using single-occupancy vehicles, which leads to:
High fuel expenses
Traffic congestion near campuses
Increased carbon emissions
Lack of trusted ride-sharing options within college communities
There is no dedicated, secure ride-sharing system designed exclusively for students.
💡 Solution
UniRide provides a secure, campus-exclusive ride sharing platform where:
Students sign up using their college email ID
Drivers can post available rides
Passengers can search for rides based on route and timing
Fuel costs are automatically split
Safety ratings ensure trust and accountability
This creates an affordable, safe, and eco-friendly commuting solution for college campuses.
🎯 Key Features
✔ College Email Verification (Only authenticated students allowed)
✔ Ride Posting (Route, time, seats available)
✔ Smart Ride Search & Matching
✔ Route Optimization
✔ Automatic Cost Splitting
✔ Safety Ratings & Feedback System
✔ Open-Source & Customizable for Universities
