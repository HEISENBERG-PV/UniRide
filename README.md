# 🚗 UniRide – Open Source Campus Ride Sharing Platform

UniRide is an open-source ride sharing platform designed specifically for college communities. It connects students travelling along similar routes, allowing them to share rides, split fuel costs, and reduce daily commuting expenses.

---

## 📖 Problem Statement

Many students commute daily using single-occupancy vehicles, which leads to:

- High fuel expenses  
- Traffic congestion around campuses  
- Increased carbon emissions  
- Lack of a secure and trusted ride-sharing system for students  

There is currently no dedicated platform built exclusively for verified college students.

---

## 💡 Solution

UniRide provides a secure, campus-exclusive ride sharing system where:

- Students register using their college email IDs  
- Drivers can post rides with route, time, and seat availability  
- Passengers can search and book rides based on destination and schedule  
- Fuel costs are automatically split among participants  
- Safety ratings promote trust and accountability  

This creates an affordable, eco-friendly, and community-driven commuting solution.

---

## 🎯 Key Features

- ✔ College Email Verification (Only authenticated students allowed)
- ✔ Post and Manage Rides
- ✔ Search and Match Rides by Route & Time
- ✔ Route Optimization
- ✔ Automatic Cost Splitting
- ✔ Driver & Passenger Safety Ratings
- ✔ Open-Source and Customizable for Universities
